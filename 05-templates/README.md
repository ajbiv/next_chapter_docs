# Templates

This folder contains reusable templates for consistent documentation across the project.

## Available Templates

- **Meeting Notes**: Structured meeting note template
- **Project Plan**: Project planning template
- **Requirements Document**: Requirements gathering template
- **Decision Record**: Architecture Decision Record (ADR) template
- **Process Documentation**: Process analysis template
- **Business Case**: Business justification template

## Usage

1. Copy the template file to the appropriate folder
2. Rename it according to the naming convention
3. Fill in the template sections
4. Remove any unused sections

## Creating New Templates

When creating new templates:
- Use clear section headers
- Include placeholder text in `[brackets]`
- Add relevant tags at the bottom
- Include a brief description at the top

# Meetings & Decisions

This folder contains meeting notes, decision records, and communication logs.

## Contents

- **Meeting Notes**: Structured notes from meetings and discussions
- **Decision Records**: Architecture Decision Records (ADRs) and key decisions
- **Action Items**: Follow-up tasks and accountability tracking
- **Communication Logs**: Important email threads and chat discussions
- **Review Sessions**: Code reviews, design reviews, and retrospectives

## Naming Convention

Use dates and descriptive titles:
- `YYYY-MM-DD-meeting-project-kickoff.md`
- `YYYY-MM-DD-decision-database-choice.md`
- `YYYY-MM-DD-review-sprint-retrospective.md`

## Meeting Note Template

```markdown
# Meeting: [Title]
**Date**: YYYY-MM-DD
**Attendees**: [List of attendees]
**Duration**: [Start - End time]

## Agenda
- [ ] Item 1
- [ ] Item 2

## Discussion
[Meeting discussion points]

## Decisions Made
- Decision 1
- Decision 2

## Action Items
- [ ] Task 1 - @assignee - Due: YYYY-MM-DD
- [ ] Task 2 - @assignee - Due: YYYY-MM-DD

## Next Steps
[What happens next]
```
